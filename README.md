# OrangeHRM Test Cases

This repository contains **manual test cases** for the 
[OrangeHRM Demo site](https://opensource-demo.orangehrmlive.com/web/index.php/auth/login).

## Modules Covered
- Login
- Admin / User Management
- PIM (Employee Management)
- Time / Attendance

## Columns in Test Cases
- TC ID
- Module
- Test Scenario
- Preconditions
- Test Steps
- Expected Result
- Status
- Priority
